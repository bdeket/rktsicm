*notes* : (listof any/c)

procedure

  note : any/c
Add a [note] to the *notes* list.

procedure

(clear-notes!) → void

Reset the *notes* list.

procedure

(show-notes) → void

Show the notes that were in the *notes* list before the last call to (clear-notes!)
